# Public content system

Website copy is composed from reviewed data in `src/content.mjs` and reusable
rendering functions in `src/components.mjs`. Help articles and selected industry
landing pages are generated from those central sources, while each route keeps a
unique title, description and useful page purpose.

`seo-content-map.json` is a planning artefact, not a publishing queue. An entry
must receive factual review, genuinely useful content and the relevant release
approval before it becomes a public route. Do not mass-generate thin pages.

All demonstrations use the fictional Northstar Home Services / Willow & Pine
Studio / INV-DEMO-1042 example. Never replace those examples with live or private
account data.
